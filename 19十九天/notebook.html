函数的定义
        function 函数名(形参) {
            函数体：函数的代码：
            var b = 11; //这是局部变量，不会在函数之外被使用，生命周期在函数结束时候结束
            c = 12;     //当没有用var定义变量时，系统自动定义一个全局变量，全局变量可以整个页面中被使用，生命周期在页面关闭后结束
        
            return 22;  //返回值，函数默认结果是undefined,此时函数的值就是22，return还会终止函数的执行 console.log(b);不被编译
            console.log(b);
        }
        函数的使用
        函数名(实参);    //函数的实参和形参的个数是允许不同的,当实参大于形参时多出的实参相当于没加入函数
        arguments//js里面提供了一个在函数中专用的伪数组，获取所有实参的对象：arguments;不能在函数外面使用
        
        写函数的步骤
        看需求，分析要实现过程，先把实现过程写出来
        分析过程里面，看哪些数据是可能产生变化的，把这些可能产生变化的变成参数
        把不会变化的过程作为函数体
        按照函数的语法写代码
        考虑是否需要修改函数的返回值

        函数表达式
        var 变量名 = function(参数){函数体} //不可被预解析
        function 变量名(参数){函数体}   //可以被预解析

        预解析——先把函数和变量的声明提升到当前作用域的最前面，而函数的调用和变量的赋值还在原来的位置
        js代码是从书写到执行的过程，是一门解释型的语言，解释一行，执行一行，如果执行到某一行，后面的语言就不再执行
        但是在一行一行解析执行之前，会进行一次预解析
        

        自动调用函数
        (function(){
            函数体
            })();

        全局作用域：在JS中任何地方都可以使用的有效范围，而在全局距定义的变量叫做全局变量
        局部作用域：只在函数中有效的范围，而在局部中定义的变量叫做局部变量

        建议使用局部变量，避免全局污染，造成全局变量相互覆盖

        匿名函数
            没有名字的行数
                function (){}
                在JS中才，不允许匿名函数单独存在，一般使用在以下地方
                -函数只是使用一次，后面再也不用的时候
                -函数表达式
                -事件
                -自调用函数

        自调用函数
            自己主动调用的函数
                (function(){})();   
                可以解决命名重复覆盖问题 
                多个自调用函数之间，需要分号隔开
                自调用函数如果使用了非匿名函数，也被无法调用   

        函数，本质也是一种数据类型function，所以函数也可以当做参数

        文字数组或者数字数组.sort(函数名);排序

        回调函数
        以参数的形式传入另外一个函数的函数，然后在另一个函数的特定时机进行调用的函数
        有一个函数a，作为函数b的参数，在函数b的特定时机，进行调用

        短路运算
        &&和||的真正运算法则
        ||  如果左边是true，整个逻辑运算结果完全取决于左边，所以直接得到左边，
            如果左边是false,整个逻辑运算的结果就完全取决于右边，直接返回右边。
        &&  如果左边是true，整个逻辑运算结果完全取决于右边，所以直接得到右边，
            如果左边是false，整个逻辑运算的结果就完全取决于左边，直接返回左边。

        形参和实参的关系，参数在函数中的赋值不会影响到外部的变量
