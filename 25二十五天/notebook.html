        事件对象
        事件对象其实就是记录了事件被触发时候，包含的一些事件相关的信息，把这些信息用一个对象整体管理起来。

        事件对象常用属性
        e.属性

        | 事件对象属性      | 描述                                           | 备注                         |
        | ----------------- | ---------------------------------------------- | ---------------------------- |
        | type              | 事件类型                                       |                              |
        |                   |                                                |                              |
        | target            | 事件目标                                       |                              |
        | srcElement        | 事件目标                                       |                              |
        |                   |                                                |                              |
        | clientX / clientY | 基于浏览器的可视区域，鼠标坐标值               | 可配合固定定位，基于窗口定位 |
        | pageX / pageY     | 基于整个页面，页面滚动有关，鼠标在页面的坐标值 | 可配合绝对定位，基于页面定位 |
        |                   |                                                |                              |
        | keyCode           | 获取键盘编码                                   |                              |
        | which             | 获取键盘编码                                   |  回车键编码13                              |

       
        事件对象常用方法
        - event.stopPropagation() 阻止事件触发后传播(阻止冒泡)
        - event.preventDefault() 取消默认行为，如a标签的跳转

        字符串.trim()   字符串修剪首尾空格

        2、event.target.className ==="item"

            箭头函数    多用于回调函数  里面的this永远指向上一级的作用域
            (参数列表)=>{函数体}
            如果函数只有一个参数可以简写成  形参 =>{函数体}
            如果函数体只有一句可以简写成    (参书列表)=> 函数语句
            如果只有一个参数一句函数体  形参=>函数语句
